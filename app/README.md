### Gradient Descender
Under construction....

It will be a GUI for deep learning and data-analysis use with system checks.

GUI can be seen with:

`poetry run invoke start`

Testing:
`poetry run invoke test`
`poetry run invoke coverage`

Pylint:
`poetry run invoke pylinter`
